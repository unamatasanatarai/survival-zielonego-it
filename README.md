Opisane tu dokumenty stanowią instrukcje do **absolutnego minimum** radzenia sobie z funkcjonalnościami, programami linuxa. 

**Przypominam** absolutne minimum. [Resztę sobie sam poszukasz ;)](https://startpage.com)
